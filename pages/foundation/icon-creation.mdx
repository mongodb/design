# Icon creation guide

The LeafyGreen system icons are a clear and concise set of glyphs designed to
be minimal in form and express a variety of use-cases for all digital
products. [Learn more about how to use Icons in
product](https://www.mongodb.design/component/icon/guidelines/") .

- Icons should represent real world things
- <>
    Icons should not represent overly complex things, like
    `CalendarEventJulyFourth`.
  </>
- Refer to this guide and existing icons while building your icon.
