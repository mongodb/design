Menus are used when a user needs to take an action.

Menus are not [select inputs](https://www.mongodb.design/component/select/guidelines/).
In a select input a selection is being made. In a menu a user takes an action
(like copy/delete a property or navigate to a different section/screen).

Menus can be triggered in many ways, such as via an ellipsis, or with a
button. Avoid using plain text or a link as a trigger for a menu.

  imageURL={`${CDN}/images/new-guidelines/menu/menu-guidelines-2@3x.png`}

Menus come default and large size. For menus that have more items use the
default size menu. For menus that have nested items use the large size.

Menus can also act as a buffer to help prevent misclicks, and are often used
where a user would expect that action to live, like near a table label for
deleting a column.
