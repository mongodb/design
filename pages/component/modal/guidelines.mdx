  Base Modal
</Subtitle>

  Use a concise title to inform the user of what action they are about to take.

  the modal, unless it’s an alert modal that doesn’t require users to take
  action. Make sure the primary button copy corresponds to the action that the
  user is taking. For example, if the title says, “Generate something” the
  primary button should say “Generate” or “Confirm”.
</Body>
